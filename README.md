# 📚 Projeto Django - Guia para Iniciantes

Este projeto foi criado com o objetivo de aprender os conceitos básicos do Django, um framework web poderoso em Python.

---

## 🚀 Sobre o Projeto

Este é um projeto simples utilizando o framework Django, ideal para quem está começando no desenvolvimento web com Python.

---

## ⚙️ Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/ThiagoLimaSanto/python-django-begin.git
cd seu-projeto
```

### 2. Crie um ambiente virtual

```bash
python -m venv venv
```

### 3. Ative o ambiente virtual

* Windows:

```bash
venv\Scripts\activate
```

* Linux/Mac:

```bash
source venv/bin/activate
```

### 4. Instale as dependências

```bash
pip install -r requirements.txt
```

---

## ▶️ Executando o Projeto

```bash
python manage.py runserver
```

Acesse no navegador:

```
http://127.0.0.1:8000/
```

---

## 🧩 Funcionalidades

* Estrutura inicial de projeto Django
* Configuração de rotas
* Exemplo de view simples
* Integração com banco de dados SQLite3

---

## 🧠 Conceitos Aprendidos

* MVT (Model-View-Template)
* Rotas (URLs)
* Views
* Migrações de banco de dados

---

## 🗄️ Banco de Dados

Para aplicar as migrações:

```bash
python manage.py migrate
```

Criar um superusuário:

```bash
python manage.py createsuperuser
```

Acessar painel admin:

```
http://127.0.0.1:8000/admin/
```
